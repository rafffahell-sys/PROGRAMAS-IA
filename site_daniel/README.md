# Site Daniel Fernandes Cardoso

Site institucional premium para apresentacao profissional de Daniel Fernandes Cardoso.

## Preview local

Abra o arquivo `index.html` no navegador ou rode um servidor local simples dentro desta pasta:

```bash
python3 -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000/
```
