📋 Instruções

Cole os dados abaixo, um registro por linha, com campos separados por ponto e vírgula (;).
Formato: Matricula ; Valor Bruto ; Qtd Meses ; Competência (MM/AAAA) ; Previdência ; Pensão
Exemplo: 123456 ; 15000,00 ; 12 ; 03/2025 ; 1500,00 ; 0
A primeira linha pode ser cabeçalho (será ignorada automaticamente).

📋 Tabelas IRRF Utilizadas

O sistema seleciona automaticamente a tabela correta com base na competência informada. No modo RRA, a base de cálculo mensal é obtida dividindo a base total pelo número de meses, e o imposto mensal resultante é multiplicado de volta pelos meses para obter o total devido.
